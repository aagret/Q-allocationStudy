# Q-allocationStudy

## Project Goal
Quantitative study and analysis of two-asset allocation strategies.

## Usage
HTML-based research tool for analyzing allocation strategies between two assets.

### Running the Application
Open the main HTML file in a web browser:
```bash
open index.html
```

### Features
- Two-asset allocation analysis
- Strategy backtesting
- Performance visualization
- Risk-return optimization

### Requirements
- Modern web browser
- Historical asset data
